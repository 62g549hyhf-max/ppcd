# PCCD v0.6

v0.5までの機能を保持し、研究機能と用語支援を追加した統合版です。

## 追加
- PCCD Academy（競技用語辞典）
- 用語のその場注釈
- 事実 / 考察 / 仮説の表示分離
- Evidence（根拠）表示
- Confidence（信頼度）表示
- 「受からない」を具体化した段階式質問
- 対戦ログから次の研究候補を提案
- 研究テーマ保存と対戦ログの関連強化

## 更新方法
GitHubリポジトリ直下へ以下4ファイルをアップロードしてください。

- index.html
- style.css
- script.js
- README.md

同名ファイルは上書きで問題ありません。
