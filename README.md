# PCCD Prototype v0.5

v0.4までの機能を保持し、以下を追加した統合版です。

## 追加
- 4画面ナビゲーション
- ポケモン詳細 Decision View
- 役割、Build、採用理由、懸念点
- 3タップ式の構築相談
- 改善候補の自動整理
- 研究テーマ保存
- 10秒対戦ログ
- localStorageによるiPhone内保存

## 更新方法
GitHubリポジトリ直下に以下4ファイルをアップロードしてください。

- index.html
- style.css
- script.js
- README.md

既存ファイルと同名なら上書きされます。
